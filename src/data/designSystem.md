## Компоновка

### Формат страницы
- **Ориентация:** Landscape (горизонтальная)
- **Пропорции:** 16:9 (≈ 1440 × 810px в PDF-просмотрщике)
- **SPA viewport:** 100vw × 100vh

### Хедер (постоянный, все страницы)
```
┌─────────────────────────────────────────────────────────────────────────────┐
│ [◆ KMF]     (пусто)          [⌂ содержание] [интегрированный [год.отчет] 2025] [kmf.kz] │
│                                             [❤ Название текущего раздела         ]      │
└─────────────────────────────────────────────────────────────────────────────┘
```
- Высота хедера: ~48–56px
- Логотип слева: красные ромбы ◆◆ + «KMF» жирным
- Правая часть: pills с навигацией
- «годовой отчет» внутри pill «интегрированный ... 2025»: красный фон pill внутри светлой
- Текущий раздел: отдельная pill ниже с красным ❤

### Футер (постоянный)
```
┌──────────────────────────────────────────────────────────────────┐
│ [←] [→]                                              [номер]    │
└──────────────────────────────────────────────────────────────────┘
```
- Высота футера: ~40–48px
- Стрелки: красные кружки (~32px) с белыми стрелками
- Номер: серый текст справа

### Сетка контента

**Большинство слайдов — 2-колоночная сетка:**
```
┌──────────────┬──────────────────────────────────────┐
│ ЛЕВАЯ 40%   │ ПРАВАЯ 60%                           │
│ (фото или   │ (текст + карточки)                   │
│  текст)     │                                      │
└──────────────┴──────────────────────────────────────┘
```

**KPI Dashboard (слайд 3) — 3-колоночная сетка, 2 ряда:**
```
┌──────────────┬──────────────┬──────────────┐
│ KPI 1        │ KPI 2        │ KPI 3        │
│ (клиенты)    │ (провизии)   │ (структура)  │
├──────────────┼──────────────┼──────────────┤
│ KPI 4        │ KPI 5        │ KPI 6        │
│ (портфель)   │ (NPL)        │ (ROA/ROE)    │
└──────────────┴──────────────┴──────────────┘
```

**ESG структура (слайд 8) — 3-колоночная:**
```
┌──────────────┬──────────────┬──────────────┐
│ 01 Стратег.  │ 02 Управл.   │ 03 Операц.   │
│ (светлый)    │ (зелёный)    │ (светлый)    │
└──────────────┴──────────────┴──────────────┘
```

### Отступы
- Внешние поля страницы: 32–40px (горизонтальные), 16px (вертикальные под хедером)
- Gap между колонками: 24–32px
- Padding внутри карточек: 24px
- Gap между строками в KPI-сетке: 16–20px

- Горизонтальная линия: 1px solid #E5E7EB

### Фоновые паттерны
- Страницы с фото: фото занимает всю левую часть, без clip/mask
- Цветные фоновые блоки: нет текстур, только flat цвет
- Градиент на концепт-странице: лёгкий тёплый градиент (кремово-персиковый) в правой части `#FFF8F0` → `#FFE8E8`

## Стиль графиков

### Общие правила

- **Фон графика:** прозрачный / белый фон карточки
- **Сетка:** да, очень лёгкая `strokeDasharray="3 3"` цвет `rgba(0,0,0,0.07)`
- **Оси:** без рамки (axisLine={false}), только tickLine={false}
- **Подписи X:** год (2023/2024/2025), 12px, #717182
- **Подписи Y:** процент или число, 11px, #717182

### Bar Chart (столбцы)

```
barSize: 32–40px
barRadius: [4, 4, 0, 0]  (скругление только сверху)
fill: url(#barGradient)   (вертикальный градиент сверху вниз)
  - top: rgba(232,32,42,0.90)
  - bottom: rgba(232,32,42,0.20)
```

### Line / ComposedChart

```
strokeWidth: 2px
stroke: #1A1A2E (тёмная линия для secondary метрик)
stroke: #E8202A (красная линия для primary)
dot: false (точки не показаны)
activeDot: { r: 4, fill: #E8202A }
```

### Area под линией

```
fill: url(#areaGradient)
  - top: rgba(232,32,42,0.15)
  - bottom: rgba(232,32,42,0.00)
```

### Stacked Bar (горизонтальный)

```
layout: "horizontal"
barSize: 28–32px
barRadius: 0 (без скруглений)
Цвета сегментов:
  - Потребительское: #E8202A + pattern (штриховка или solid)
  - Розничная торговля: #F5C542
  - Сельхоз: #B0B7C3
```
*Примечание: в PDF видно, что красный сегмент имеет тонкую горизонтальную штриховку/текстуру*

### Donut Chart

```
innerRadius: 60%
outerRadius: 85%
paddingAngle: 2
strokeWidth: 0 (без обводки между сегментами)
```

### Легенда

```
позиция: снизу или справа в зависимости от графика
маркер: небольшой цветной прямоугольник (4px × 4px) или кружок (6px)
шрифт: 12px, #3D3D4E
```

### Подписи значений на барах

```
position: top
fontSize: 14–16px
fontWeight: 700
fill: #1A1A2E
```

## Логотип KMF

```
◆◆ KMF
```
- Два ромба: красный (#E8202A) + жёлтый (#F5C542)
- Текст «KMF» рядом, жирный, тёмный (#1A1A2E) или белый (на тёмном фоне)
- Размер в хедере: ~28–32px высота
